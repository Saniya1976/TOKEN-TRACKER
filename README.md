# 🪙 Token Tracker X

A sleek, modern cryptocurrency dashboard for tracking real-time prices, market trends, and historical data. Powered by the CoinGecko API, this app provides a premium experience for monitoring the global crypto market.

---

## 🚀 Core Features

- **Real-Time Market Tracking**: View the top 100 cryptocurrencies by market cap with live price updates.
- **Trending Carousel**: A visual, auto-scrolling display of trending coins represented by circular logos.
- **Advanced Coin Details**: Deep-dive into any coin to see high/low stats, circulating supply, and market cap rank.
- **Interactive Price Charts**: Analyze historical price movements using beautiful area charts with multiple time ranges (24H, 7D, 1M, 1Y).
- **Global Search & Sort**: Instantly find coins by name/symbol and sort the market table by price, volume, or change percentage.
- **Multi-Currency Support**: Seamlessly toggle between **USD ($)** and **INR (₹)** across the entire application.
- **Responsive & Dark Mode**: A premium "Deep Dark" aesthetic designed for maximum readability and a high-end feel.

## �️ Tech Stack

- **Frontend**: React 19 & Vite 6
